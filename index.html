Information Security Risks

System Architecture

Actors & Their Roles

SMCS serves four distinct user types with tailored access levels.

1

Soldier

Messages family, accesses missions, triggers SOS alerts, receives commander updates.

2

Commander

Creates and manages missions, monitors troops, reviews all SOS alerts in real time.

3

Family Member

Secure messaging portal only — completely blocked from missions and operational data.

4

System Admin

Manages user roles, security policies, and monitors system-wide health.
